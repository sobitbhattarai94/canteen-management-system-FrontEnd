# Canteen Management System (Frontend)

A modern **React + Vite** frontend project for managing a canteen. Users can browse food items, filter by category, add items to the cart, and view totals. Admin dashboard allows managing products and orders. Built with **Tailwind CSS**, **React Router**, and **React Icons**, fully responsive, with a vibrant **yellow theme** representing hunger and food energy.

### User Side

- Filter food items by category (Snacks, Meals, Drinks, Desserts)
- Add items to cart with quantity controls
- View total price and checkout (frontend only)

### Admin Dashboard
- Sidebar navigation for dashboard pages
- Manage products: view, add, delete (frontend state only)
- Add new product with name, price, category, and image URL
- View dummy orders and order details
- Statistics cards for total products, orders, and revenue

### UI & UX
- Yellow-themed vibrant interface
- Smooth hover effects on buttons and cards
- Responsive grid and layout
- Reusable components (FoodCard, StatsCard, Navbar, CartModal)
